TextFilter for PAQ by P.Skibinski

1. It is based on WRT 4.4 (with improvements from PASQDA).
2. It works even without dictionaries (uses EOL coding, capital conversion, etc.)
3. The best compression performance is achieved with dictonaries supplied in /TextFilter directory
   (the same dictionaries are needed for decompression).
4. You can easily add new dictionaries (modifications in PAQ aren't necessary).
